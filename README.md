# C++ Backend Development Lab
Практические мини‑проекты для прокачки бэкенд‑навыков на C++.

Здесь я создаю и тестирую решения, чтобы отточить ключевые навыки:

   - **архитектура бэкенда**: обработка запросов с использованием потоков и асинхронных операций;
   - **сетевое взаимодействие**: реализация клиентов и серверов с использованием протоколов TCP/UDP и HTTP;
   - **работа с базами данных**: интеграция C++ приложений с хранилищами данных, использование `ORM`;
   - **инструменты и DevOps**: автоматизация сборки с помощью `CMake`, контейнеризация в `Docker`, настройка логирования и мониторинга производительности.

## Использование

### Сборка проекта
   ```bash
   mkdir build && cd build
   cmake ..
   cmake --build .
   ```

### Запуск тестов
   ```bash
   ./tests/unit_tests
   ```
   или
   ```bash
   ctest --verbose
   ```
